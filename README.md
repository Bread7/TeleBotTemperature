# Telegram Bot to send temperature WIP (Automated and Manual)

## Development Setup
Install necessary packages
```
$ yarn install
```
Copy .env.example to personal .env and change variables accordingly
```
$ cp .env.example .env
```
To run application
```
$ yarn start
```
